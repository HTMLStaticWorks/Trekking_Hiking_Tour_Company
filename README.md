# TrekX - Premium Trekking & Hiking Tour Company

Welcome to the **TrekX** project! This is a modern, responsive, and visually stunning web application for a premium trekking and hiking tour company. It features a rich UI designed to captivate adventure seekers.

## Features

- **Fully Responsive Design**: Optimized layouts for Desktop, Tablet, and Mobile devices.
- **Dynamic Theming**: Built-in Dark and Light mode toggle.
- **RTL Support**: Built-in Right-to-Left layout toggle for internationalization.
- **Modern UI/UX**: Includes parallax scrolling, glassmorphism effects, and micro-animations.
- **SEO & Accessibility**: Semantic HTML, meta descriptions, and ARIA labels.

## Pages Included
1. `index.html`: Main landing page with hero banner, floating stats, and popular destinations.
2. `home2.html`: Alternative landing page layout.
3. `about.html`: Company information, team, and mission.
4. `services.html`: Detailed list of offered trekking and camping services.
5. `packages.html`: Available trekking packages and pricing plans.
6. `destinations.html`: Stunning gallery of global hiking destinations.
7. `gallery.html`: Photo gallery and blog grid.
8. `contact.html`: Contact form, FAQs, and location details.
9. `signup.html`: User registration and login forms.

## Technology Stack
- **HTML5**: Semantic structure and forms.
- **CSS3**: Vanilla CSS for styling, Flexbox/Grid for layout, Custom Properties (Variables) for theming.
- **Vanilla JavaScript**: Lightweight DOM manipulation for theme toggling, RTL switching, and mobile menu interactions.
- **Font Awesome**: Iconography.

## How to Run
Simply open `index.html` in any modern web browser to view the site. No local server or build tools are required, though using a live server (like VS Code Live Server) is recommended for the best experience.
